1. Welche Daten muss ich täglich eintragen?

Tragen Sie Ihre aktuellen Werte zu Gesundheit, Ernährung, Aktivität, Stimmung und Umgebung ein.

2. Muss ich alle Felder ausfüllen?

Nein, Sie können auch nur die für Sie relevanten Felder ausfüllen.

3. Wie kann ich meine Stimmung angeben?

Klicken Sie auf einen der Emoji-Buttons, um Ihre aktuelle Stimmung auszuwählen.

4. Was passiert, wenn ich einen Fehler gemacht habe?

Sie können Ihre Angaben jederzeit anpassen und erneut speichern.

5. Werden meine Daten automatisch gespeichert?

Ihre Einträge werden nach dem Speichern im System gesichert und können später eingesehen werden.
